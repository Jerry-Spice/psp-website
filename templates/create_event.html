Instructions

This page will build an event based on the information you pass in.

First Box The first box is the name of the event. This should ideally be something short and quick to read so that brothers can identify it quickly on the calendar. For longer pieces of information continue to the second box.

Second Box The second box is for the event's description. Write some general details about the event including potential information like meeting place, items needed by members, etc. This box is going to be formatted using the Markdown formatting langauge. This lanugage is similar to HTML and will allow you to format the description however you want. Note markdown also accepts raw HTML and CSS in case you want to style the description in a fancy way.

Third and Fourth Boxes The third and fourth boxes are time and date accordingly these will appear on the event's card in the calendar and it will affect how it appears on the calendar. Make sure you enter the date correctly and double check the calendar to make sure it was created properly.

Fifth Box (dropdown) This dropdown determines what point will be awarded for brothers that go to the event. It defaults to "N/A" because that is more often then not what will be awarded for events like weekly chapter.

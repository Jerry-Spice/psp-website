Instructions

This page will build an event based on the information you pass in.

First Box The first box is the title of the announcement. This will appear larger than the description text below.

Second Box The second box is for the announcement's description. Write some general details about the announcement including information like meeting place, items needed by members, etc. This box is going to be formatted using the Markdown formatting langauge. This lanugage is similar to HTML and will allow you to format the description however you want. Note markdown also accepts raw HTML and CSS in case you want to style the description in a fancy way.
